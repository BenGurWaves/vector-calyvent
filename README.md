# VECTOR — PDF to Figma Vectors

A client-side PDF to SVG vector extraction tool for designers. Drop a PDF, extract clean layered vectors, and paste directly into Figma.

## Features

- **Zero server costs**: All processing happens in the browser
- **Privacy-first**: Files never leave your device
- **15MB free tier**: Process smaller files at no cost
- **Clean SVG output**: Production-ready vectors ready for Figma

## Usage

1. Drop a PDF file onto the drop zone
2. Wait for vector extraction to complete
3. Click "Copy SVG Vector Data to Clipboard"
4. Paste directly into Figma (Cmd+V)

## Tech Stack

- PDF.js for client-side PDF parsing
- SVGGraphics for vector path extraction
- Vanilla JavaScript (no frameworks)
- Cloudflare Pages for deployment
